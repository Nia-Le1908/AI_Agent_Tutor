##  XỬ LÝ SỰ CỐ THƯỜNG GẶP
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

# ❌ Lỗi "No module named ..."
#    → Kiểm tra đã activate venv chưa: venv\Scripts\activate
#    → Chạy lại: pip install -r requirements.txt

# ❌ Lỗi "GEMINI_API_KEY not set"
#    → Kiểm tra file .env có đúng ở thư mục gốc không
#    → Key phải bắt đầu bằng "AIza..."

# ❌ Lỗi "Database is locked" hoặc "no such table"
#    → Chạy lại: python init_db.py

# ❌ Lỗi "Ollama connection refused"
#    → Mở terminal riêng, chạy: ollama serve
#    → Kiểm tra: ollama list

# ❌ Lỗi FAISS / "faiss_index.bin not found"
#    → Chưa nạp tài liệu. Chạy: python embedder.py
#    → Đảm bảo có file PDF/DOCX trong data/raw_docs/

# ❌ Streamlit mở nhưng trắng / lỗi import
#    → Dùng đúng Python trong venv:
#       venv\Scripts\python.exe -m streamlit run app.py

# │   └── ai_tutor_v5.db     ← SQLite database
