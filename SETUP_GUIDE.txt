# Cập nhật: 2026-05-11

# Mục tiêu:
# - Bạn có thể clone repo -> cài deps -> chạy Streamlit.
# - Có 2 chế độ:
#   A) DEMO (không cần API key): chạy Exercise/Dashboard với mock DB.
#   B) FULL (đầy đủ Chat RAG + tạo câu hỏi): cần DEEPSEEK_API_KEY + build FAISS.
#
# Lưu ý bảo mật:
# - File .env KHÔNG được commit lên GitHub (repo đã ignore).
# ================================================================


================================================
0) Yêu cầu
================================================

- Python 3.10+ (khuyến nghị 3.10/3.11)
- Git (tuỳ chọn nếu bạn clone)
- Windows PowerShell hoặc Terminal


================================================
1) Lấy source code
================================================

Nếu dùng Git:
	git clone https://github.com/Nia-Le1908/AI_Agent_Tutor.git
	cd AI_Agent_Tutor

Nếu bạn copy folder thủ công:
	mở terminal tại thư mục gốc project (nơi có requirements.txt)


================================================
2) Tạo venv + cài thư viện
================================================

--- Windows PowerShell ---

	python -m venv .venv

	# Nếu bị chặn activate script:
	Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned

	.\.venv\Scripts\Activate.ps1
	python -m pip install --upgrade pip
	pip install -r requirements.txt


--- macOS / Linux (bash/zsh) ---

	python3 -m venv .venv
	source .venv/bin/activate
	python -m pip install --upgrade pip
	pip install -r requirements.txt


================================================
3) Tạo file .env (khuyến nghị copy từ .env.example)
================================================

--- Windows PowerShell ---
	Copy-Item .env.example .env

--- macOS / Linux ---
	cp .env.example .env

Sau đó mở .env và chỉnh tuỳ theo chế độ bạn chạy.


================================================
4A) CHẾ ĐỘ DEMO (không cần API key) — chạy được ngay
================================================

Mục tiêu: mở UI và dùng Exercise/Dashboard ngay với dữ liệu mock.
Chat/Admin-Generate sẽ KHÔNG hoạt động nếu không có API key.

1) Tạo mock DB và mock questions:
	python generate_mock_data.py

2) Mở file .env và đặt:
	DB_PATH=mock_data/mock_db.sqlite

3) Chạy app:
	streamlit run app.py

Truy cập: http://localhost:8501


================================================
4B) CHẾ ĐỘ FULL (Chat RAG + tạo câu hỏi) — khuyến nghị
================================================

Bạn cần:
	- DEEPSEEK_API_KEY (bắt buộc để Chat và tạo câu hỏi)
	- Có tài liệu PDF/DOCX để build FAISS (bắt buộc để Chat không lỗi retrieval)

1) Mở file .env và set tối thiểu:

	# --- LLM (bắt buộc) ---
	DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
	DEEPSEEK_MODEL=deepseek-chat

	# --- Storage ---
	DB_PATH=data/ai_tutor_v5.db
	FAISS_INDEX_PATH=vector_store/faiss_index.bin
	LOG_PATH=logs/app.log

	# --- RAG tuning ---
	CHUNK_SIZE=256
	CHUNK_OVERLAP=50
	TOP_K=3
	EMBEDDING_MODEL_NAME=all-MiniLM-L6-v2
	DEBUG_RAG_CONTEXT=false

2) Khởi tạo SQLite database:
	python init_db.py

3) Nạp tài liệu cho RAG (bắt buộc để Chat chạy):
	- Copy ít nhất 1 file .pdf hoặc .docx vào: data/raw_docs/
	- Sau đó chạy:
			python embedder.py

	Kết quả sẽ tạo:
		- vector_store/faiss_index.bin
		- vector_store/chunks_metadata.json

4) Chạy app:
	streamlit run app.py

5) Tạo câu hỏi để Exercise có dữ liệu:
	- Vào sidebar "Admin Panel (Tạo bài tập)" trong app
	- Nhập chủ đề + độ khó + số lượng -> bấm tạo
	- Câu hỏi sẽ được insert vào SQLite (bảng questions)


================================================
5) Lỗi thường gặp (Troubleshooting)
================================================

1) "No module named ..."
	- Chưa activate venv hoặc cài deps
	- PowerShell:
			.\.venv\Scripts\Activate.ps1
			pip install -r requirements.txt

2) Chat bị lỗi "DEEPSEEK_API_KEY is required"
	- Bạn đang chạy FULL nhưng chưa set DEEPSEEK_API_KEY trong .env

3) Chat bị lỗi "FAISS index file not found" hoặc "chunks_metadata.json not found"
	- Bạn chưa build vector store
	- Đặt PDF/DOCX vào data/raw_docs/ rồi chạy:
			python embedder.py

4) Exercise báo không có câu hỏi
	- DB chưa có dữ liệu trong bảng questions
	- Cách nhanh nhất: dùng Admin Panel trong app để generate & save vào DB

5) "Database is locked" / "no such table"
	- Chạy lại init:
			python init_db.py
